Atividades Extracurriculares

Projeto de Iniciação Científica

Universidade Estácio de Sá em conjunto com a COPPE/UFRJ | Ago/2017 a Ago/2018 Este projeto visava o desenvolvimento de um modelo matemático multicritério para análise de criticidade dos componentes de um sistema complexo. Este modelo permite identificar os elementos mais críticos do sistema, ou seja, aqueles que, na ocorrência de uma falha, impactariam mais severamente na sua função. Desta forma, é possível uma melhor organização das atividades de manutenção e uma alocação mais adequada dos recursos, que são limitados. Para exemplificar e validar o modelo, foi realizado um estudo de caso utilizando como sistema o trem de levitação magnética desenvolvido na COPPE/UFRJ, o MagLev-Cobra.
